# rCAD Utils

A collection of utility tools for CAD operations, optimized for use with Blender.

## Features

- **Extrude Along Path**: Extrude geometry along a custom path curve with full control and flexibility
- **Mirror Along Plane**: Create mirrored copies of geometry across a plane with precise alignment
- More tools coming soon...

## Installation

1. Clone this repository
2. Copy the `rCAD_utils` folder to your Blender add-ons directory
3. Enable the add-on in Blender's Preferences > Add-ons

## Usage

Each utility tool can be accessed through Blender's UI. Refer to individual tool documentation for detailed usage instructions.

## Requirements

- Python 3.9+
- Blender 3.0+
